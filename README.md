# sap-purchase-analytics
Custom SAP ABAP ALV report for purchase analytics using EKKO, EKPO, LFA1 tables.
## 📌 Description
This project is a custom SAP ABAP report developed to analyze purchase data using standard SAP tables. It provides insights into purchase orders, vendor-wise spending, and total procurement value.
## 🎯 Objective
To build an interactive ALV report that helps businesses track and analyze purchase transactions efficiently.
## ⚙️ Features
* Fetch data from SAP tables (EKKO, EKPO, LFA1)
* Calculate total purchase amount (Quantity × Price)
* Interactive ALV report display
* Sorting and filtering options
* Hotspot click functionality
* Conditional formatting (highlight high-value orders)
## 🛠️ Technologies Used
* SAP ABAP
* ALV (CL_SALV_TABLE)
## 📊 Output
The report displays:
* Purchase Order Number
* Vendor Name
* Material
* Quantity
* Net Price
* Total Amount
